# Stok Takip Web Uygulamasi

Modern, responsive stok takip sistemi. React + Node.js + SQLite.

## Kurulum

### 1. Repoyu indir
```
git clone https://github.com/navruzaligalip-cpu/stok-takip.git
cd stok-takip
```

### 2. Backend
```
cd web/backend
npm install
npm run dev
```

### 3. Frontend (yeni CMD penceresi)
```
cd web/frontend
npm install
npm run dev
```

### 4. Tarayicida ac
```
http://localhost:3000
```

## Ozellikler
- Dashboard: istatistikler ve grafikler
- Urun yonetimi (CRUD, barkod, kategori)
- Stok girisi / cikisi / iade / sayim
- Raporlar ve analiz
- Mobil uyumlu tasarim
